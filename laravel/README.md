# WEBSITE INTRODUCE

## Installation

```sh
cp .env.example .env
composer install
php artisan key:gen
php artisan migrate:fresh --seed
npm i
npm run dev
php artisan ser
```
## Laravel-filemanager
```
php artisan storage:link

Edit APP_URL in .env. (exmam: APP_URL=http://127.0.0.1:8000)
```
